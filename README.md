# Lucky Draw - PWA 應用

這是一個漸進式網頁應用（PWA），可以在 Android 和 iOS 上安裝使用。

## 功能特色

- ✅ 可安裝到主畫面
- ✅ 支援離線功能
- ✅ 跨平台（Android 和 iOS）
- ✅ 無需 App Store 審核
- ✅ 自動更新

## 如何安裝

### Android
1. 在 Chrome 瀏覽器中開啟應用
2. 點擊瀏覽器選單中的「安裝應用程式」或「加入主畫面」
3. 確認安裝

### iOS (iPhone/iPad)
1. 在 Safari 瀏覽器中開啟應用
2. 點擊分享按鈕
3. 選擇「加入主畫面」
4. 確認加入

## 專案結構

- `app.html` - 主要應用程式
- `manifest.json` - PWA 設定檔
- `service-worker.js` - 離線快取服務
- `icon-192.png` - 192x192 應用圖標
- `icon-512.png` - 512x512 應用圖標

## 開發說明

直接編輯 `app.html` 檔案即可，修改後重新整理瀏覽器即可看到變更。
